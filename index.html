Frequency Distribution
Main
table with data in ascending order usually, relating to a measurable characteristic according to individual value or a group of values of the characteristic under study
distributes total frequency to number of classes
no. of class interval x class lengths = Range
Class Limit
minimum and max value, class may contain
Min is Lower Class Limit
Max is Upper Class Limit
Class Boundary
actual limit for class interval
CL = CB in case of continuous variable (10-20, 20-30)
In discrete
0-9, 10-19, 20-29
LCB = LCL - D /2
UCB = UCL + D/2
Mid Value/Mid Point / Class Mark
LCL + UCL /2 or LCB + UCB /2
Width or size of a class interval
difference b/w UCB and the LCB
Cumulative Frequency
the number of observations less than the value or less than or equal to the class boundary
Frequency Density
Freq/Class size
Relative or percentage frequency
ratio of the class frequency to the total frequency
relative add up to 1, percentage add up to 100
